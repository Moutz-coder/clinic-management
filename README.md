# نظام إدارة العيادات الطبية وحجز المواعيد

نظام متكامل لإدارة العيادات الطبية وحجز المواعيد الإلكتروني، مصمم لسهولة الاستخدام من قبل المرضى والأطباء والإداريين.

## 🌟 المميزات

### للمريض
- **تصفح العيادات**: البحث عن العيادات والأطباء حسب التخصص والموقع
- **حجز المواعيد**: حجز مواعيد إلكترونية بسهولة وسرعة
- **تتبع المواعيد**: عرض المواعيد القادمة والسابقة وحالتها
- **المحادثات**: التواصل مع العيادات والأطباء
- **الإشعارات**: تلقي إشعارات حول المواعيد والتحديثات
- **الملف الشخصي**: إدارة المعلومات الشخصية

### للعيادة/الطبيب
- **لوحة تحكم شاملة**: إحصائيات مفصلة عن المواعيد والمرضى
- **إدارة المواعيد**: قبول أو رفض أو إلغاء المواعيد
- **إدارة المرضى**: عرض معلومات المرضى وسجلاتهم الطبية
- **السجلات الطبية**: إضافة وتعديل السجلات الطبية للمرضى
- **المحادثات**: التواصل مع المرضى
- **دخول الأطباء**: دخول سريع للأطباء باستخدام رمز PIN
- **إعدادات العيادة**: إدارة معلومات العيادة وساعات العمل

### للإدارة
- **إدارة العيادات**: الموافقة على طلبات تسجيل العيادات الجديدة
- **إدارة المستخدمين**: عرض وإدارة جميع المستخدمين
- **لوحة تحكم شاملة**: إحصائيات عامة عن النظام

## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية (Frontend)
- **React 19** - مكتبة JavaScript لبناء واجهات المستخدم
- **Vite** - أداة بناء سريعة وحديثة
- **TailwindCSS 4** - إطار عمل CSS للتصميم
- **React Router** - إدارة التنقل بين الصفحات
- **Lucide React** - مكتبة الأيقونات
- **Axios** - للاتصال بالخادم
- **date-fns** - لمعالجة التواريخ

### الواجهة الخلفية (Backend)
- **Node.js** - بيئة تشغيل JavaScript
- **Express.js** - إطار عمل ويب
- **MongoDB** - قاعدة بيانات NoSQL
- **Mongoose** - ODM للتعامل مع MongoDB
- **JWT** - للمصادقة والتأمين
- **Bcrypt** - لتشفير كلمات المرور
- **Multer** - لرفع الملفات

## 🚀 التثبيت والتشغيل

### 1. تثبيت MongoDB

#### على Windows باستخدام Chocolatey:
```bash
choco install mongodb -y
```

#### على macOS:
```bash
brew tap mongodb/brew
brew install mongodb-community
```

#### على Linux:
```bash
sudo apt-get install mongodb
```

### 2. تشغيل MongoDB

#### على Windows:
```bash
net start MongoDB
```

#### على macOS/Linux:
```bash
sudo systemctl start mongod
```

### 3. تثبيت المشروع

```bash
# استنساخ المشروع
git clone <repository-url>
cd dock

# تثبيت الاعتماديات
npm install
cd client
npm install
cd ..
```

### 4. إعداد المتغيرات البيئية

أنشئ ملف `.env` في المجلد الرئيسي:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/clinic_management
JWT_SECRET=your_super_secret_jwt_key_change_in_production
JWT_EXPIRES_IN=7d
UPLOAD_DIR=uploads
```

### 5. تشغيل المشروع

```bash
# تشغيل الخادم والواجهة معاً
npm run dev:all

# أو تشغيل كل جزء على حدة
npm run dev          # الخادم فقط
npm run client       # الواجهة فقط
```

### 6. إضافة البيانات التجريبية (اختياري)

```bash
npm run seed
```

سيتم إنشاء:
- حساب مدير: `admin@clinic.ly` / `admin123`
- حساب مريض: `patient@clinic.ly` / `patient123`
- حساب عيادة: `center@clinic.ly` / `clinic123`
- عيادة مع 3 أطباء (رمز الدخول للأطباء: `123456`)
